=== Visual Studio 2012 ASP.NET Web サイト テンプレート

Microsoft® Visual Studio® 2012 ASP.NET ウェブ サイト テンプレートは、マスターページに Infragistics コントロールと正しく動作しないスタイルを含みます。

*回避方法*

マスター ページのデフォルトのスタイルが Infragistics コントロールと競合を発生します。競合を防止するには、デフォルト マスター ページで以下のコードを削除する必要があります。

*HTML の場合:*

[source,html]
----
<% if (false) { %><%--Designer support--%>
<link href="~/Content/Site.css" rel="stylesheet" />
<% } %>
----
