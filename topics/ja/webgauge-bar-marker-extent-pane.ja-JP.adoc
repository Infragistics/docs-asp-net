= 範囲ペイン

[範囲] ペインは、バー マーカーの幅を決定します。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.lineargaugebarmarker~innerextent.html[内側]"]  -- この値を 0 から 100 の整数に設定します。内側の値は、リニア ゲージの左側からバーの開始までの距離です。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.lineargaugebarmarker~outerextent.html[外側]"]  -- この値を 0 から 100 の整数に設定します。外側の値は、リニア ゲージの左側からバーの終わりまでの距離です。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.lineargaugebarmarker~startextent.html[開始]"]  -- この値を 0 から 100 の整数に設定します。開始値はバー マーカーの開始値です。
