= マウスをドラッグする間に画像をスクロール

WebImageViewer™ コントロールによってエンド ユーザーはマウスをドラッグする間に画像をスクロールできます。Continuous スクロール アニメーションを使用して画像がスクロールされます。

デフォルトで、 pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.listcontrols.webimageviewer~enabledragscrolling.html[EnableDragScrolling]"]  プロパティは True に設定されています。

以下のコード スニペットは EnableDragScrolling を False に設定します。

image::images/WebImageViewer_Scroll_Images_While_Dragging_the_Mouse_01.png[WebImageViewer_Scroll_Images_While_Dragging_the_Mouse_01]

*Visual Basic の場合：*

----
' EnableDragScrolling を False に設定します。
Me.WebImageViewer1.EnableDragScrolling = false
----

*C# の場合：*

----
// EnableDragScrolling を False に設定します。
this.WebImageViewer1.EnableDragScrolling = false;
----
