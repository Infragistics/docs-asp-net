= WebPageStyler について

WebPageStyler コンポーネントによって、プロジェクト全体でアプリケーション スタイリングをオンにすることなく、特定の ASPX ページでアプリケーションスタイリングを使用することができます。このコンポーネントの利点は、同じテーマとスタイルの代わりにアプリケーション全体で異なるテーマとスタイルを使用できることです。WebPageStyler コンポーネントによって、アプリケーションでのより高い柔軟性と汎用性が可能となります。異なるスタイルセットを作成して、プロジェクト全体のさまざまな ASPX ページに実装することができます。
