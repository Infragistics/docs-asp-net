= ランタイムで StyleSets を変更

アプリケーション スタイリングの非常にユニークな機能は、ランタイムにスタイルセットを変更する機能です。この機能は複数のエンド ユーザーがひとつのアプリケーションを使用する場合に有益で、各エンド ユーザーは異なるスタイル プレファレンスを持つことができます。ドロップダウン リストで簡単な選択をするだけで、それぞれの選択のスタイルセットを指定できます。

以下のサンプル コードは、この概念を示しており、ランタイムにエンド ユーザーがスタイルセットを適用できます。選択して使用可能なスタイルセットを含むドロップダウン リストを使用します。

*Visual Basic の場合：*

----
Private Sub Page_Load(ByVal sender As System.Object, _
  ByVal e As System.EventArgs) Handles MyBase.Load
	Dim settings As AppStylingManager = AppStylingManager.Settings
	settings.StyleSetName = "Pear"
End Sub
----

*C# の場合：*

----
private void Page_Load(object sender, System.EventArgs e)
{
	AppStylingManager settings = AppStylingManager.Settings;
	settings.StyleSetName = "Pear";
}
----
