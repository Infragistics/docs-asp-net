= コントロールの参照

クライアント側で WebCalendarView™ コントロールを参照するためには 2 つの方法があります。WebSchedule™ コントロールのすべては、ig_getWebControlById 関数を使用し、クライアントでそれにアクセスするためにコントロールの ID を渡すことができます。WebCalendarView では特に、コントロールを参照するために ig_getWebCalendarViewById 関数を使用できます。

次のコードはこのコントロールの参照方法を示します。

*JavaScript の場合*

----
var calendarView = ig_getWebCalendarViewById("WebCalendarView1");
----
