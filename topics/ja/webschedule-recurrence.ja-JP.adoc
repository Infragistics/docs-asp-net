= 定期的

WebSchedule は繰り返し予定をサポートします。これらの予定は、繰り返し予定が動作する方法を制御する特定の規則セットに従います。[繰り返し] ダイアログで選択することによって、またはコードで予定を作成している場合には予定で設定することによって、これらの規則を有効または修正できます。
