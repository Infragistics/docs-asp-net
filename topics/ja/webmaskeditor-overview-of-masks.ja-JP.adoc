= マスクの概要

[options="header", cols="a,a,a,a"]
|====
|文字|説明|サンプル マスク|可能な入力値

|0
|数値 (0から9、入力は必須です。プラス [+] およびマイナス [-] 符号は許可されません)。
|(000) 000-0000
|(206) 555-0248

|9
|数値またはスペース（入力は必須ではありません）。
|9999
|2 0

|#
|数値またはスペース（入力は必須です）。
|####
|2000

|L
|文字 (A から Z、入力は必須です)。
|>LL00000-0000
|DB51392-0493

|?
|文字 (A から Z、入力はオプションです)。
|>L????L?000L0
|MAY R 452B7

|A
|文字または数値（入力は必須です）。
|(000) AAA-AAAA
|(206) 555-TELE

|a
|文字または数値（入力はオプションです）。
|aaaaaaaaaa
|Item 123

|&
|任意の文字またはスペース（入力は必須です）。
|ISBN 0-&&&&&&&&&0
|ISBN 1-55615-507-7

|C
|任意の文字またはスペース（入力はオプションです）。
|CCCCCCCCCCC
|Hello There

|. , : ; - /
|これらの文字は、1000 単位、小数点、および日時値を分離するために使用されます。使用される実際の文字は、Microsoft Windows のコントロールパネルに指定されている地域設定によって異なります。
|0,000.00
|1,999.45

|<
|この文字に続くすべての文字は小文字に変換されます。後続の < 文字は、小文字の機能を実際に切り替えます。たとえば、マスク< td>
|>L<??????????????
|Maria

|>
|この文字に続くすべての文字は大文字に変換されます。後続の > 文字は、大文字の機能を実際に切り替えます。たとえば、マスク >LL>LL>LL は、大文字の機能を 2 番目の LL 文字で無効にしますが、3 番目の LL 文字では元に戻します。
|>L<??????????????
|Joe

|\
|この文字に続く文字は、次の \ 文字が現れるまで、マスクのリテラル部分として取り扱われます。
|A\AB\B\\\##\99
|xABB\# 91

|====
