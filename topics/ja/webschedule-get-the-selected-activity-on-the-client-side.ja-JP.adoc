= 選択されたアクティビティを取得（クライアント側）

WebDayView および WebMonthView はすべて、クライアント側に getSelectedActivity と呼ばれるメソッドがあり、選択した pick:[asp-net="link:{ApiPlatform}webui.webschedule{ApiVersion}~infragistics.webui.webschedule.activity.html[アクティビティ]"] にアクセスするために使用できます。このメソッドはクライアント側でアクティビティを削除または更新するときに使用できます。

以下のコードはこのメソッドの使用を示しています。null エラーを回避するためにアクティビティが選択されていることを確認するためのテストに注意してください。

*JavaScript の場合:*

----
function GetSelectedActivity()
{
	var webDayView = ig_getWebControlById("WebDayView1");
	var activity = webDayView.getSelectedActivity();
	if(activity != null)
		alert(activity.getSubject());
	else
		alert("You must select an Activity first");
}
----
