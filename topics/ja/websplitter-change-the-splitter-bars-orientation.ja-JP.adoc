= 分割バーの方向を変更

WebSplitter™ コントロールは以下の 2 つの方向モードをサポートします。

* Horizontal
* Vertical

デフォルトで、Orientation プロパティは Vertical に設定されます。

以下の例のコードは、Orientation プロパティを Horizontal に設定します。

*Visual Basic の場合：*

----
' Orientation を Horizontal に設定します。
Me.WebSplitter1.Orientation = Infragistics.Web.UI.LayoutControls.SplitterOrientation.Horizontal
----

*C# の場合：*

----
// Orientation を Horizontal に設定します。
this.WebSplitter1.Orientation = Infragistics.Web.UI.LayoutControls.SplitterOrientation.Horizontal;
----
