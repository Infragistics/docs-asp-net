= WebDataProviders について

データ アクセス レイヤに配置されている WebSchedule データ プロバイダは、WebScheduleInfo 内の  pick:[asp-net="link:{ApiPlatform}webui.webschedule{ApiVersion}~infragistics.webui.webschedule.activity.html[Activity]"]  オブジェクトと  pick:[asp-net="link:{ApiPlatform}webui.webschedule{ApiVersion}~infragistics.webui.webschedule.resource.html[Resource]"]  オブジェクトを、データストア内のテーブルおよび行にバインディングを実行します。

データ プロバイダ クラス モデルは、拡張性を考慮して作成されているので、データ プロバイダを拡張して他のデータソースや拡張オブジェクト モデルを処理できます。
