= アセンブリ

アセンブリは、共通言語ランタイム（CLR）のコア部分です。これは、アプリケーションを実行するために CLR が必要とするタイプとリソースの集合で、通常はひとつの実行可能ファイル（.exe）または DLL（dynamic-link library）の形で格納されています。アセンブリは、配備、バージョンコントロール、セキュリティのパーミッションを含むアプリケーションの重要な側面を制御します。

以下の表は、{ProductName} 製品スイートに含まれるすべての ASP.NET アセンブリの一覧です。この表には、各アセンブリに含まれるASP.NETコントロールとコンポーネントも指定され、アセンブリのすべてのリソース文字列のリストを含むリソース文字列ヘルプトピックへのクロスリファレンスが含まれています。
