= 値と単位

[値と単位] ペインによって、ゲージの針が指し示す値を設定することができます。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.gaugemarker~value.html[値]"]  -- この値を 0 から 100 の整数に設定します。この値は、針がスケールのどこを指し示すのかを決定します。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.gaugemarker~precision.html[精度]"]  -- 項目の値を 0.00 から 100.00 の値に設定します。この値はマーカーの「丸めの」数です。たとえば、精度値が 0.5 に設定されると、マーカーの値は常に 0.5 の倍数にスナップします。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedle~widthmeasure.html[単位]"]  -- この値は、針の測定の単位を指定します。この値は、リニア ゲージで針を作成している場合には使用できません。
