= ボックス注釈

注釈は、追加情報を追加する、またはゲージに視覚的な手がかりを提供するために使用されます。ボックス注釈は矩形で、端が丸められたり、その中にオプションのテキストが付けられている場合があります。

たとえば、温度計を表すためにリニア ゲージを作成した場合、測定の単位を示すためにテキストを [摂氏] に設定して注釈を作成することができます。
