= パフォーマンス

WebSchedule は ViewState を使用せずに動作するように設定されています。したがって、ViewState を無効にしたい場合でも、WebSchedule コントロールは継続して動作します。

さらに、WebSchedule コントロールはスマート ナビゲーションを利用するように設計されています。

* link:webschedule-enabling-smart-callbacks-ajax-for-webschedule-controls.html[WebSchedule コントロールのスマート コールバック (AJAX) を有効にする]
* link:webschedule-limiting-appointment-dates-and-times.html[予定の日時を制限]
* link:webschedule-stop-the-webscheduleviews-from-synchronizating.html[WebSchedule Views の同期を停止]
