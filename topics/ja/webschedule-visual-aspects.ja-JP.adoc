= 視覚的側面

WebSchedule™ コントロールには、設定およびカスタマイズ可能な複数の視覚的な側面があります。これには、コントロールに表示される画像が含まれます。たとえば、ビュー コントロールのヘッダーに表示されるナビゲーション画像はカスタム画像に設定できます。アラームを使用して、予定に表示される画像をカスタマイズすることも可能です。

ビュー コントロールで使用されている選択スタイルを固有のカスタム スタイルにカスタマイズすることも可能です。

プリセット、Cascading Styles Sheets を使用する、またはコントロールのスタイルのプロパティを変更することによって、コントロールの表示スタイルを変更できます。
