= WebHtmlEditor のツールバーのダイアログボックス ボタン

ユーザーがアプリケーションをクリックすると、 pick:[asp-net="link:{ApiPlatform}webui.webhtmleditor{ApiVersion}~infragistics.webui.webhtmleditor.toolbardialogbutton.html[ToolbarDialogButton]"]  クラスのインスタンスがポップアップ ダイアログ ボックスを開きます。ユーザーがタスクを遂行するためにさらに入力する必要があるときにこのボタンを使用します。エンド ユーザーは、ダイアログ ボックス ボタンをクリックしてから、タスクの遂行を取りやめるために開いたダイアログ ボックスを閉じることができます。表示されるダイアログ ボックスは、各 ToolbarDialogButton オブジェクトの  pick:[asp-net="link:{ApiPlatform}webui.webhtmleditor{ApiVersion}~infragistics.webui.webhtmleditor.htmlboxdialog~caption.html[Caption]"]  プロパティと  pick:[asp-net="link:{ApiPlatform}webui.webhtmleditor{ApiVersion}~infragistics.webui.webhtmleditor.htmlboxdialog~text.html[Text]"]  プロパティを使ってカスタマイズできます。ダイアログ ボタンを使用する操作には、ユーザーがパレットから色を選択（ハイライト、テキストの前景色など）したり、関数のパラメータを指定（検索や置換）したりする例が挙げられます。

次の表に、WebHtmlEditor コントロールのツールバーの CategoryName ボタンのリストを示します。各ボタンの説明は、 link:webhtmleditor-list-of-webhtmleditors-toolbar-buttons.html[WebHtmlEditor のツールバー ボタンのリスト]をお読みください。
