= つまみの操作を設定

WebSlider™ で 2 つのつまみを有効にすると、各つまみが互いに作用する方法を調整できます。ThumbsInteractionMode 列挙体は、さまざまなシナリオで WebSlider のつまみの動作を処理するための広範なオプションを提供します。

つまみはいくつかの異なる方法で作用します:

* Free - つまみは互いを自由に通過してスライドできます。
* Push - つまみをもう一方のつまみの横にスライドすると、互いにプッシュします。
* Lock - つまみは互いを通過してスライドすることも互いをプッシュすることもできません。

以下の例のコードは、WebSlider のつまみの操作モードを Push に設定する方法を示します。

image::images/WebSlider_Set_Thumb_Interaction_01.png[]

*Visual Basic の場合：*

----
Me.WebSlider1.ThumbsInteractionMode = _ Infragistics.Web.UI.EditorControls.SliderThumbsInteractionMode.Push
----

*C# の場合：*

----
this.WebSlider1.ThumbsInteractionMode = Infragistics.Web.UI.EditorControls.SliderThumbsInteractionMode.Push;
----
