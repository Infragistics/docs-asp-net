= バック アンカー レイアウト タブ

Gauge コントロールの  pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedle~backanchor.html[Back Anchor]"]  プロパティは、Needle  pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugescale~markers.html[Marker]"]  プロパティと関連して使用されます。バック アンカーはアンカーの真下に描画されます。

[プロパティ] パネルの [バック アンカー レイアウト] タブを使用して、ゲージ上のバック アンカーをスタイルできます。ゲージ エクスプローラでバック アンカーをクリックする（[ラジアル ゲージ] > [スケール] > [マーカー] > [針] > [バック アンカー]）、またはインタラクティブなプレビュー領域でラジアル ゲージのバック アンカーをクリックするのいずれかによって、このタブにアクセスできます。

タブは以下の 3 つのペインに分割されています。

* link:webgauge-back-anchor-extent-pane.html[範囲]
* link:webgauge-back-anchor-radius-pane.html[半径]
* link:webgauge-back-anchor-spoke-pane.html[スポーク]
