= シンプル グラデーション ブラシ エレメント

ゲージのプロパティを特定の方法でひとつの色で開始して別の色にフェードインするようにしたい場合には、ブラシ タイプを SimpleGradient に設定します。SimpleGradient ブラシ エレメントを選択すると、開始色、終了色、グラデーション スタイルを指定できます。以下を使用してシンプル グラデーション ブラシ エレメントをゲージのプロパティに適用することができます。

* link:webgauge-apply-brush-elements-using-the-gauge-designer.html[ゲージ デザイナ]
* link:webgauge-apply-brush-elements-at-design-time.html[ゲージ コレクション エディタ]
* link:webgauge-apply-brush-elements-at-run-time.html[コード]

以下のスクリーンショットは、以下の設定で目盛盤の BrushElement プロパティを SimpleGradientBrushElement に設定した状態のラジアル ゲージを表示します。

* 開始色 -- 192, 0, 0
* 終了色 -- 黒
* グラデーション スタイル -- 垂直
