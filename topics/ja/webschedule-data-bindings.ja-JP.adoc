= データ バインディング

WebSchedule™ データ プロバイダーは、Data Binding プロパティを公開します。これによって Activity や Resource などの WebSchedule オブジェクトのプロパティに応じて選択の列名を設定することができます。これらのバインディング メンバー プロパティを使用して、カスタム データ モデルにアクセスする時に WebSchedule データ プロバイダーが検索する列名を変更します。

ストアド プロシージャが事前にコンパイルされているため、動的 SQL で動作するのはデータ バインディング機能のみです。
