= その他ペイン

[その他] ペインは、ゲージがラジアル ゲージでスケールできるように使用されます。

*その他* -- この値を True に設定すると、ラジアル ゲージが円に表示して傾かないようにデフォルトで常に行ってきたように、ソースの境界を四角形に制限します。　このプロパティのデフォルト値はラジアル ゲージの場合を除き False です。したがって、ゲージがあり、その下にあるラジアル ゲージで適切にスケールしたい場合には、[四角形] プロパティを True に設定すると、境界がラジアル ゲージに沿ってスケールします（[四角形] プロパティはデフォルトで True に設定されます）。
