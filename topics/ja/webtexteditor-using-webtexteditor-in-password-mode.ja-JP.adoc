= パスワード モードで WebTextEditor を使用

WebTextEditor™ はエンドユーザーがエディターに入力する内容をスタンバイしている他の人々が見ることができないようにパスワード モードに簡単に設定できます。

*注：* パスワード モードで WebTextEditor に入力したテキストは暗号化されないので、他の機密データのように保護する必要があります。

パスワード モードで WebTextEditor を使用するには、コントロールの  pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.editorcontrols.webtexteditor~textmode.html[TextMode]"]  プロパティを Password に設定する必要があります。Microsoft® Visual Studio® [プロパティ] ウィンドウを使用する、または以下のコードを使用することによって、TextMode プロパティを Password に設定できます。

*Visual Basic の場合：*

----
WebTextEditor1.TextMode = TextBoxMode.Password
----

*C# の場合：*

----
WebTextEditor1.TextMode = TextBoxMode.Password;
----
