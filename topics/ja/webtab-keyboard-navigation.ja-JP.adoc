= キーボード ナビゲーション

WebTab™ は、キーボードを使用したタブの簡単なナビゲーションをサポートします。

* 左矢印 – アクティブ タブを左に移動します。
* 右矢印 – アクティブ タブを右に移動します。
* Home – 最初のタブをアクティブにします。
* End – 最後のタブをアクティブにします。
* Space/Enter – アクティブ タブを選択します。

*注：* WebTab は、コントロールの  pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.layoutcontrols.webtab~enableactivation.html[EnableActivation]"]  プロパティを True に設定した場合に限って、キーボードに応答します。
