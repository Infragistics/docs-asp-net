= ゲージ ラベルについて

ラベルはスケールの点を表すために使用される数値またはテキスト値です。スケール上の目盛と関連付けられており、マーカーが指し示す値をエンド ユーザーが読み取りやすくすることができます。

たとえば、温度計を表すためにリニア ゲージを作成した場合、ラベル値で表された温度を読み取ることができます。
