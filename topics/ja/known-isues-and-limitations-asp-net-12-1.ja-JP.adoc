=== グリッド アクセシビリティ準拠第 508 条

グリッドは、1973 年リハビリテーション法の第 508 条第 1194 項第 22 節の規則をサポートしていません。

(g) 行ヘッダーと列ヘッダーは、データ テーブル用に識別される必要があります

=== IIS 7 がインストールされた Windows Server 2008 SP2 での画像の問題

{ProductName} が、IIS 7 がインストールされた Windows Server 2008 SP2 にインストールされている場合、画像は表示されません。フォルダー `C:\Inetpub\wwwroot\aspnet_client\Infragistics\Images` に、web.config ファイルが自動的に生成され、アクセスが自動的にブロックされます。

=== IIS 7 がインストールされた Windows Server 2008 SP2 での画像の問題

開発中は、この問題を解決するため、images フォルダーに自動的に生成された web.config ファイルを削除してください。
