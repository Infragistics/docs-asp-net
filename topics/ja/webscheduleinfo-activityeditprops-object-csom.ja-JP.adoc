= ActivityEditProps オブジェクト

== ActivityEditProps

ActivityDialogEdit クライアント側イベントで引数として渡されるオブジェクト。

=== ActivityEditProps プロパティ

[cols="a,a"]
|====
|*AllDayEvent* 
|[アクティビティ] ダイアログに関連付けられた Element と Value を含むオブジェクト。

|*Description* 
|[アクティビティ] ダイアログに関連付けられた Element と Value を含むオブジェクト。

|*DisplayInterval* 
|[アクティビティ] ダイアログに関連付けられた Element と Value を含むオブジェクト。

|*document* 
|[アクティビティ] ダイアログのドキュメント オブジェクト。

|*Duration* 
|アクティビティの期間の Value を含むオブジェクト。

|*EnableReminder* 
|[アクティビティ] ダイアログに関連付けられた Element と Value を含むオブジェクト。

|*EndDate* 
|[アクティビティ] ダイアログに関連付けられた Element、Value、および WebDateChoooser オブジェクトを含むオブジェクト。

|*EndTime* 
|[アクティビティ] ダイアログに関連付けられた Element、Value、および WebDateChoooser オブジェクトを含むオブジェクト。

|*Importance* 
|[アクティビティ] ダイアログの重要性に関連付けられた WebToolbar、Value、および Toolbar Items を含むオブジェクト。

|*Location* 
|[アクティビティ] ダイアログに関連付けられた Element と Value を含むオブジェクト。

|*ShowTimeAs* 
|[アクティビティ] ダイアログに関連付けられた Element と Value を含むオブジェクト。

|*StartDate* 
|[アクティビティ] ダイアログに関連付けられた Element、Value、および WebDateChoooser オブジェクトを含むオブジェクト。

|*StartTime* 
|[アクティビティ] ダイアログに関連付けられた Element と Value を含むオブジェクト。

|*Subject* 
|[アクティビティ] ダイアログに関連付けられた Element と Value を含むオブジェクト。

|*window* 
|[アクティビティ] ダイアログに関連付けられた window オブジェクト。
