== 既知の問題

Linux で実行する時、ファイルの大文字小文字の識別がアプリケーションに障害を発生させる場合があります。たとえば、ig_datagrid.css としてページの先頭部分で ig_dataGrid.css が参照されますが、これは Windows ベースの Web サーバーでは動作しますが、Linux ボックスでは障害が発生します。回避策のひとつは、小文字でファイル名を変更することです。
