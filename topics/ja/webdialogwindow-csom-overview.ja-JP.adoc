= CSOM の概要

WebDialogWindows™ コントロールは、クライアント側の JavaScript プログラミング環境に堅牢なオブジェクト モデルを提供します。オブジェクト モデルは、十分な機能を備えた JavaScript オブジェクトで構成され、開発者はサーバー側にポストバックしなくても、ダイアログ ウィンドウに関連する重要な機能単位をプログラムできます。
