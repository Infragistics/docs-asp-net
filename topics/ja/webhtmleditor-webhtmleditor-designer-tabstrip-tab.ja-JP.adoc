= WebHtmlEditor デザイナ（タブストリップ タブ）

タブストリップでは、WebHtmlEditor の初期画面としてデザイン ビュー（アクティブなツールバーが付いたリッチ WYSIWYG 編集モード）を表示するか、HTML ビュー（エンドユーザーが HTML で記述できる強力なマークアップ エディタ）を表示するかを StartupMode 値で指定できます。変更した場合は、[適用] をクリックして確定してください。

image::images/WebHtmlEditor_Using_the_Property_Builder_of_WebHtmlEditor_09.jpg[]

このタブでは、デフォルトのテキスト ラベルと、タブでラベルの左側に表示されるアイコンを変更することもできます。同じ変更をコード ビハインドのプログラミングにより設定するための手順を追った説明は、 link:webhtmleditor-customize-the-tab-strip-of-webhtmleditor.html[「タブストリップのカスタマイズ」]のトピックを参照してください。
