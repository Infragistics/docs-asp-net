= ダイアログ ウィンドウをモーダルにする

エンド ユーザーからの入力が必要で入力なしに続行できない場合があります。このような場合、Windows フォームのモーダル ダイアログ ボックスと同様に、WebDialogWindow™ コントロールのダイアログ ウィンドウをモーダル ダイアログ ボックスとして表示できます。エラーが発生しエンド ユーザーがエラーを修正後でなければ続行できない場合にこれは役に立ちます。ダイアログ ウィンドウをモーダル ダイアログ ボックスにするには、 pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.layoutcontrols.webdialogwindow~modal.html[Modal]"]  プロパティを True に設定する必要があります。ダイアログ ウィンドウがモーダルになると、このウィンドウが再度消されるまでエンド ユーザーは Web ページの任意の部分を操作できなくなります。

以下のコードは、ダイアログ ウィンドウをモーダル ダイアログ ボックスに変える方法を例示します。

*Visual Basic の場合：*

----
' エンド ユーザーが続行する前に 
' 処理するように強制します。
Me.WebDialogWindow1.Modal = True
----

*C# の場合：*

----
// エンド ユーザーが続行する前に 
// 処理するように強制します。
this.WebDialogWindow1.Modal = true;
----
