= キーボード ナビゲーション

WebExplorerBar™ は、キーボード ナビゲーションをサポートしており、エンドユーザーがグループと項目間を簡単かつ素早くナビゲートできます。

* 上矢印 / 下矢印 – グループと項目を上/下に移動します。
* 左矢印 - 選択したグループまたは項目を縮小します。
* 右矢印 - 選択したグループまたは項目を展開します。
* Enter – NavigateUrl を起動します。
