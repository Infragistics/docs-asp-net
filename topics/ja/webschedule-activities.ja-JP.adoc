= アクティビティ

アクティビティはアプリケーションのスケジュールに必須です。WebSchedule™ は、抽象的なの基本クラスのインスタンスによって、特定の開始日時に発生するイベントのモデルを作成します。アクティビティは瞬間的アクティビティまたは長期的に実行するアクティビティ（終日および複数日イベント）にすることができます。

ユーザーが操作するアクティビティの最も一般的な例はで、これは件名、場所、説明を持つアクティビティです。WebSchedule ビュー要素はさまざまな方法で予定を提示します。右の例では、WebDayView™ と WebMonthView™ コントロールが同じ日のスケジュールを表示しています。

WebScheduleInfo は、すべての関連付けられたビューが共有するコレクション内の表示されたアクティビティを含み、アクティビティを保存してデータ ストアから取得するためにデータ プロバイダーと連動します。
