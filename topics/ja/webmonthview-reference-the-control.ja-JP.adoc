= コントロールの参照

クライアント側で WebMonthView™ コントロールを参照するためには 2 つの方法があります。すべての WebSchedule™ コントロールは、ig_getWebControlById 関数を使用し、クライアントでそれにアクセスするためにコントロールの ID を渡すことができます。WebMonthView では特に、コントロールを参照するために ig_getWebMonthViewById 関数を使用できます。

次のコードはこのコントロールの参照方法を示します。

*JavaScript の場合*

----
var monthView = ig_getWebMonthViewById("WebMonthView1");
----
