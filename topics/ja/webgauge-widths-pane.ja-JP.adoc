= 幅ペイン

[幅] ペインは、ラジアル ゲージまたはリニア ゲージ上の各目盛の太さを決定します。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugescaletickmarkappearance~startwidth.html[開始]"]  -- この値を 0 から 100 の整数に設定します。この値は目盛の開始点の幅を設定します。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugescaletickmarkappearance~endwidth.html[終了]"]  -- この値を 0 から 100 の整数に設定します。この値は目盛の終了点の幅を設定します。
