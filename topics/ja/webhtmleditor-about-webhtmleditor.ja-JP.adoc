= WebHtmlEditor について

WebHtmlEditor は、Web アプリケーションでエンド ユーザーが使用する What You See Is What You Get（WYSIWYG）編集ウィンドウで豊富な編集機能を提供します。

* link:webhtmleditor-webhtmleditor-toolbar.html[ツールバー] -- 編集およびアップロード操作用のボタンとドロップダウン リストが含まれています。
* link:webhtmleditor-webhtmleditor-text-window.html[テキスト ウィンドウ] -- エンド ユーザーが内容を編集してプレビューできます。
* link:webhtmleditor-webhtmleditor-tab-strip.html[タブ ストリップ] -- エンド ユーザーがデザインビューと HTML ビューを切り替えるためのタブが含まれています。
