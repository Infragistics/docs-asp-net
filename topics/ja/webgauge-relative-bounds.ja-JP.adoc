= 相対バウンド

[相対バウンド] ペインは、ゲージ上の Brush 要素のサイズと位置を決定します。

X -- この値は Gauge コントロールの左側からの Brush 要素の距離を決定します。

Y -- この値は Gauge コントロールの上側からの Brush 要素の距離を決定します。

Width -- この値は Brush 要素の幅を決定します。

Width -- この値は Brush 要素の高さを決定します。

Units -- この値は Brush 要素の測定の単位を指定します。Brush 要素は、ピクセルまたは Gauge コントロールのパーセントで測定できます。
