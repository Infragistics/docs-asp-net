= プロパティ パネル

ゲージ デザイナの [プロパティ] パネルは、現在選択されているゲージのプロパティを表示します。[プロパティ] パネルの詳細は、 link:webgauge-about-properties-panel.html[「プロパティ パネルについて」]を参照してください。

[プロパティ] パネルはタブを使用してデザインされています。これらのタブによって、ゲージのプロパティを編集できます。タブは以下のカテゴリーに分られます。

* link:webgauge-changing-the-layout-of-your-webgauge.html[WebGauge のレイアウトの変更]
* link:webgauge-changing-the-appearance-of-your-webgauge.html[ゲージの外観の変更]
* link:webgauge-changing-the-annotations-on-your-webgauge.html[WebGauge 上の注釈の変更]
* link:webgauge-applying-effects-to-your-webgauge.html[効果を WebGauge に適用]
