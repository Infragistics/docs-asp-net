= WebGroupBox アクセシビリティの遵守

弊社の ASP.NET コントロールおよびコンポーネントはすべて、1973 年のリハビリテーション法第 508 条第 1194 部 22 条を遵守しています。以下は、第 1194 部 22 条の 16 の規則と各規則を遵守する方法を含んだ表です。場合によって、プロパティを設定する必要があります。それ以外の場合には、コントロールがすべての作業を実行するため、ユーザーは何もする必要がありません。

[options="header", cols="a,a"]
|====
|規則|規則を遵守する方法

|(n)
|WebGroupBox がこのアクセシビリティの規則を満たすために、コントロールで TabIndex プロパティを設定し、グループ ボックス内に配置するコンテンツにラベルを付ける必要があります。また、コンテンツが第 508 条を遵守していることを確認する必要があります。コンテンツにラベルを付けるためには、コントロール、または起動されたときに発生するアクションを説明する HTML ラベルを各コントロールの横に配置する必要があります。
