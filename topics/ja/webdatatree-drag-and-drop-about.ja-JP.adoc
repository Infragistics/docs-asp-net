= ドラッグ アンド ドロップについて

WebDataTree のドラッグアンドドロップ機能は、ノードのドラッグ、ノードのグループを同じ WebDataTree やその他のコントロールへドロップする機能があります。この動作は 3 つのモードがあります - デフォルト、コピー、移動。 デフォルト モードは、Ctrl キーを長押しするとコピーとして動作します。そうでない場合は移動になります。選択したノードやノードの前、間、後へ挿入できます。 この場合、ドロップしたノードはドラッグアンドドロップ先のノードの子になります。 ドラッグしたノードを他のノードへ移動する間、選択したノードのドロップする場所にツールチップが表示されます。 2 つのノード間を移動すると線が表示され、その隣に「上のノードとノードの間に挿入してください」というテキストが表示されます。 サーバー サイドとクライアント サイド、ドラッグアンドドロップに関連するイベントの両方があります。詳細については、ClientSideEvents および ServerSideEvents のトピックをご参照ください。WebDataTree $$->$$ DragDropSEttings AllowDorop および EnableDragDrop に両方のプロパティを設定すると、ドラッグおよびドロップのすべての機能を使用できます。 NodeDragged サーバーサイド イベントはノードがドラッグされるたびに処理する必要はありません。 Esc キーでドラッグをキャンセルできます。または WebDataTree 領域外へノードをドラッグアンドドロップします。
