= 範囲ペイン

[範囲] ペインによって、アンカーの位置を決定できます。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedleanchor~spokedepth.html[深さ]"]  -- 値を 0 から 100 までの.整数に設定します。 この値は、ラジアル ゲージの中央とスポーク間の間隔です。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.radialgaugeneedleanchor~spokesweep.html[スイープ]"]  -- 値を 0 から 360 までの整数に設定します。 この値は各スポークの角度測定です。 スイープ値が 100 以下の場合にのみ、アンカーの「スポーク」外観が表示されます。
