= WebCurrencyEditor のカルチャーを設定

WebCurrencyEditor™ によって、コントロールの  pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.editorcontrols.webnumericeditor~culture.html[Culture]"]  プロパティを設定することによって通貨の表示方法に関する特定のカルチャーを設定できます。WebCurrencyEditor はデータをフォーマットするために CultureInfo クラスに依存します。デフォルトでシステムに設定されている現在のカルチャーを使用します。

Microsoft® Visual Studio® [プロパティ] ウィンドウを使用する、または以下のコードを使用することによって、Culture プロパティを設定できます:

*Visual Basic の場合：*

----
'使用したいカルチャーまたはロケールを指定します
 Dim c As New System.Globalization.CultureInfo("en-GB")
'CultureInfo オブジェクトを WebCurrencyEditor のカルチャーに指定します
 WebCurrencyEditor1.Culture = c
----

*C# の場合：*

----
//使用したいカルチャーまたはロケールを指定します
System.Globalization.CultureInfo c = new System.Globalization.CultureInfo("en-GB");
//CultureInfo オブジェクトを WebCurrencyEditor のカルチャーに指定します
WebCurrencyEditor1.Culture = c;
----
