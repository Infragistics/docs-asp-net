= ペインのサイズを設定

WebSplitter™ コントロールはすべてのペインをピクセルでサイズ設定できます。ペインのパーセントを指定する場合、WebSplitter はランタイムにパーセントをピクセルに変換してその数を公開します。デザインタイムにパーセント値を指定し、エンド ユーザーがペインをリサイズする場合、WebSplitter はブラウザ ウィンドウと周りのパネルの既知の幅に基づいて幅のパーセントを再計算します。WebSplitter は更新したパーセント値を公開します。
