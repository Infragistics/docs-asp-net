= Web ページの更新制限

WebSchedule アプリケーションのユーザーは、ブラウザ内でページの更新オプションを使用することで、特定の状況で予期しない結果が生じることに注意が必要です。

特に、新しい予定を追加直後に [更新] (

image::images\WebSchedule_Related_Topic_Browser_Refresh_Issue_01.png[]

) ボタンをクリックすると、その予定はシステムに 2 度追加されます。ブラウザは警告メッセージ ボックスを表示し、追加した予定情報がサーバーに再送信されることを警告します（以下参照）。

Web ブラウザの [更新] ボタンを使用しないでください。

image::images\WebSchedule_Related_Topic_Browser_Refresh_Issue_02.png[]
