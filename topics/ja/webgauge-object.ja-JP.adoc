= WebGauge オブジェクト

== UltraWebGauge

ゲージ CSOM オブジェクト。ig_getWebGaugeById 関数を呼び出してパラメータとしてゲージの ID を指定することで取得できます。

=== WebGauge メソッド

==== refresh

非同期にゲージ画像を更新します。クライアント ブラウザに新しい画像を送信する前に AsyncRefresh イベントがサーバー上で発生します。
