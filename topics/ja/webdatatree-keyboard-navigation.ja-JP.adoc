= キーボード ナビゲーション

WebDataTree™ は非常に簡単にキーボードを使用したノードのナビゲーションをサポートします。

* *上下左右* -- 上矢印および下矢印キーはアクティブなノードをひとつずつ進めます。左矢印および右矢印キーは子ノードの表示を切り替えます（ツリー ノードを縮小または展開）。
* *Enter* -- Url がある場合その Url にナビゲートします。
* *スペース* -- ナビゲートせずにノードを選択します。
* *Tab* -- ページ上で次のプレーンの DOM 要素またはコントロールにフォーカスを移動します。
* *F2* -- F2 キーを押すとノードの編集が有効な場合にはノードは編集モードになります。Esc を押すとノードの編集はキャンセルされ、Enter または Tab を押すと新しいテキストをコミットします。
* *Home* -- 最初のノードに移動します。
* *End* -- 最後のノードに移動します。
* *Shift* -- 矢印キーでノードを同時に選択する間 Shift キーを押し続けることによって複数選択（連続したノード）が実行されます。
* *Ctrl* -- 矢印キーでノードをナビゲートする間同時に Ctrl キーを押し続けることによって複数選択（非連続のノード）が実行され、次にスペース キーを押すことで選択または選択を解除することができます。
