= WebSchedule の汎用データ プロバイダーのデータ モデル

WebScheduleGenericDataProvider™ コンポーネントによって、ユーザーが選択する任意のデータ ソースにバインドすることができます。ただし、すべての機能が正しく動作するには、データ ソースに必要となる特定のフィールドが含まれなければなりません。さらに、自動データ バインディングを行うには、データ フィールドは特定の名前を持つ必要があります。

アクティビティ、バリアンス、およびリソースのデータ スキーマについての詳細は、以下のリンクをクリックしてください。

* link:webschedule-generic-activityvariance-data-model.html[汎用的なアクティビティ/バリアンス データ モデル]
* link:webschedule-generic-resource-data-model.html[汎用的なリソース データ モデル]
