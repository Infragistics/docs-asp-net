= キーボード ナビゲーション

WebDropDown™ はキーボード ナビゲーションをサポートします。

* *Tab* - ドロップダウンにフォーカスを設定します。
* *ALT + AccessKey (ALT + SHIFT + AccessKey)* - ドロップダウンでフォーカスします
* *Esc* - ドロップダウンを自動的に閉じて、ドロップダウン コンテナが開かれた時、または開いている間に現在実行中のアニメーションをすべてキャンセルします。
* *Alt+ 下矢印* - フォーカスがドロップダウンに置かれたらドロップダウン コンテナを表示します。
* *Alt+ 上矢印* - フォーカスがドロップダウンに置かれたらドロップダウン コンテナを非表示にします。
* *下矢印と上矢印* - 項目をナビゲートして、ドロップダウン コンテナが表示されると現在の項目が自動的にアクティブ化および選択されます。エンドユーザーが *Esc* または *Enter* を押すときに限りドロップダウン コンテナは閉じられます。
* *Enter* - NavigateUrl を呼び出します。
* *スペース +Ctrl* - 複数選択が有効の時に項目を選択または選択を解除します。
* *Ctrl + 上/下* - 複数選択が有効の時に現在のアクティブな項目を変更しますが選択を修正しません（Windows Explorer の場合と全く同じように）。
* *Home* - 最初のドロップダウン項目をスクロールして選択およびアクティブ化します。
* *End* - 最後のドロップダウン項目をスクロールして選択およびアクティブ化します。
