= Dialog オブジェクト

== ダイアログ

ActivityDialogOpeningクライアント側イベントで引数として渡されるオブジェクト。

=== Dialog メソッド

[cols="a,a"]
|====
|*getContentUrl* 
|showDialog が呼び出されるときに開く URL。

|*getCaption*
|ダイアログのキャプション。

|*setCaption*
|ダイアログのキャプションを設定します。 

*パラメタ*

_文字列_ 

新しいキャプションとして表示する文字列値。

|*showDialog*
|指定した高さと幅でダイアログを表示します。 

*パラメタ*

_height_ 

整数値。 _width_ 

整数値。

|====
