= プロパティ パネル

プロパティ パネルは、アプリケーションを実際にスタイルする場所です。ロールを見つけて（スタイル エクスプローラで、またはキーボード ショートカットを使用して）、プロパティ パネルによってロールを変更し、キャンバスでこれらの変更を確認します。プロパティ パネルはデザイナとテキスト エディタの 2 つのセクションに分かれています。デザイナは、スタイルを視覚的に設定して色を選択することができる場所です。テキスト エディタは文字通り、raw CSS テキストを編集します。デザイナでのすべての変更はテキスト エディタに反映され、テキスト エディタでのすべての変更はデザイナに反映されます。

* link:webappstylist-designer.html[デザイナ] -- デザイナは、すべて変更可能なプロパティの論理的なグルーピングです。ただし、ロール ツリーでどのロールをクリックするかによって、異なるプロパティが表示されます。コントロール自体をクリックすると、修正すべきコントロール固有のプロパティが提供されますが、これらは CSS に変換しません。コントロールの下にある個々のロールをクリックすると、直接 CSS クラスに変換します。
* link:webappstylist-text-editor.html[テキスト エディタ] -- クリックするよりも入力するほうが速いと思う場合には、テキスト エディタを使用して、CSS を関連付けられたクラスに直接入力します。
