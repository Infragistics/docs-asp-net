= 標準規格のサポート

WebSchedule コントロールは複数の異なる規格をサポートしますが、そのひとつがセクション 508 の準拠です。WebSchedule コントロールも十分に Cascading Style Sheets を利用して、コントロールのスタイルで使用するためのこれらの規格をサポートします。WebSchedule コントロールも HTML 4.01 準拠です。
