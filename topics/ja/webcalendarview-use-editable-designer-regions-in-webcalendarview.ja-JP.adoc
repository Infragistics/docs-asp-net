= WebCalendarView で編集可能なデザイナ領域を使用

Visual Studio 2005（.NET Framework 2.0）では、デザインタイム テンプレートの代わりに、編集可能な領域が使用されます。この編集可能領域は、常に編集モードで表示される単なるテンプレートです。Visual Studio 2005 では、すべての Web 要素を 1 回の操作で編集可能領域へドラッグできるので、Visual Studio 2002/2003 のように編集テンプレート機能を使用する必要はありません。

WebCalendarView には、カレンダーのフッタに編集可能なデザイナ領域があります。フッタをクリックするだけで、この領域が青い境界線で囲まれます。その後、フッタでの編集を行えます。
