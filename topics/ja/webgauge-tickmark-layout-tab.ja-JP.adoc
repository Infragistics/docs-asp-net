= 目盛レイアウト タブ

Gauge コントロールの  pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.gaugescaletickmarkappearance.html[Tickmark]"]  プロパティは、ラジアル ゲージまたはリニア ゲージのスケールに値を視覚的に表示するために使用されます。2 つのタイプの目盛があります。

* 副目盛
* 主目盛

両方の目盛を構成して特定の値に「目盛」を表示することができます。主目盛は副目盛よりも通常大きいです。これらの目盛はスケールで n ごとの値を強調表示するように構成します。

[プロパティ] パネルの [目盛レイアウト] タブを使用して、ゲージ上の目盛をカスタマイズできます。ゲージ エクスプローラで [主目盛] または [副目盛] をクリックする（[ラジアル ゲージ/リニア ゲージ] > [スケール] > [主/副目盛]）、またはインタラクティブなプレビュー領域でゲージ上の目盛をクリックすることによって、このタブにアクセスできます。

タブは以下の 3 つのペインに分割されています。

* link:webgauge-extent-pane.html[範囲]
* link:webgauge-widths-pane.html[幅]
* link:webgauge-orientation-pane.html[方向]
