= 開発中に Medium Trust の下で Web アプリケーションをテストする方法とは？

以下の XML の引用で示すように、Web アプリケーション プロジェクトの Web.config ファイルの <trust> 要素を変更することによって、開発でのホスティング プロバイダーの信頼レベルをターゲットとすることができます。

[source]
----
...
<system.web>
...
      <authentication mode="Windows" />
       <trust level="Medium" />
</system.web>
...
----
