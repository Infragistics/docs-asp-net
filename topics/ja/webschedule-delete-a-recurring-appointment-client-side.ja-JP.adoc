= クライアント側の繰り返し予定を削除

既存の Recurrence オブジェクトを更新するには、繰り返しを持つ予定を最初に取得する必要があります。このトピックでは、予定のデータキーを知っていることを前提とします。繰り返しを削除するため、予定からオブジェクトを取得する必要はありません。その代わりに、Recurrence 属性を Null に設定するためには activityUpdateProps を設定するだけです。最後の手順は、WebScheduleInfo の updateActivity メソッドを呼び出すことです。

*JavaScript の場合:*

----
// この予定を Activities コレクションから取得します。
var appointment = oWebScheduleInfo1.getActivities().getItemFromKey("3522");
// activityUpdateProps の Recurrence 属性を null に設定します。
var activityUpdateProps = {Recurrence: null};
// 更新したプロパティを渡すことによってアクティビティを更新します。
oWebScheduleInfo1.updateActivity(activityUpdateProps, appointment, "");
----
