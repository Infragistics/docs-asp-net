= バー マーカー

バー マーカーは、ラジアル ゲージとリニア ゲージのスケールで開始点と終了点を持つバーとして表示されます。

たとえば、温度計を表すリニア ゲージを作成している場合には、水銀レベルを表示するためにバー マーカーを使用します。
