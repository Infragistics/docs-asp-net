= SecureImagePipe について

SecureImagePipe コントロールは、.aspx ページを介して画像をストリーミングするために 使用する ASP.NET サーバー コントロールです。このコントロールをセッションの WebChart の pick:[asp-net="link:{ApiPlatform}webui.ultrawebchart{ApiVersion}~infragistics.webui.ultrawebchart.ultrachart~deploymentscenario.html[シナリオの展開]"] オプションと併用すると、レンダリングされたチャート画像のファイルをディスクに書き込む必要がなくなり、単に画像ファイルを ASP.NET セッション状態で格納すれば、画像ファイルがクライアントにストリームされます。
