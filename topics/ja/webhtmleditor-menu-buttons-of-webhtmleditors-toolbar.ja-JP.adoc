= WebHtmlEditor のツールバーのメニュー ボタン

ToolbarMenuButton クラスのインスタンスは、アプリケーションのエンド ユーザーがクリックするとコンテキスト メニューを開きます。これらのボタンは、ユーザーが 1 つのパラメータに値を入力するタスクの編集と表示をサポートしていますが、エンド ユーザーが選択できる値の数が比較的小さいときに使用します。このコンテキスト メニューに適した操作の例としては、ドキュメントのズームに 50%、100%、200% などの値を選択する場合が挙げられます。

次の表に、WebHtmlEditor コントロールのツールバーの CategoryName ボタンのリストを示します。各ボタンの説明は、 link:webhtmleditor-list-of-webhtmleditors-toolbar-buttons.html[WebHtmlEditor のツールバー ボタンのリスト]をお読みください。
