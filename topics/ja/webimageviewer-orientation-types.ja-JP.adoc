= 方向のタイプ

WebImageViewer™ は水平（デフォルト）と垂直に方向を設定することができます。 pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.listcontrols.webimageviewer~orientation.html[Orientation]"]  プロパティが Horizontal に設定され、Infragistics Application Styling Framework によって提供されるようにデフォルト スタイルを使用すると、画像を水平にスクロールできます。これは、[左] または [右] ナビゲーション ボタンをクリックする、または画像を単に左右にクリック アンド ドラッグするのいずれかが可能であることを意味します。Orientation プロパティが Vertical に設定されると、ナビゲーション ボタンは WebImageViewer の上部と下部に配置されます。画像を上および下からクリック アンド ドラッグすることもできます。

*注:* 方向に可能な限り最高の方法を選択して画像を表示するために WebImageViewer コントロールをリサイズするのは賢明です。
