= WebGrid は Safari ブラウザをサポート

{ProductName} 2008 Volume 2 から開始しましたが、WebGrid™ コントロールは Apple® Safari™ Web ブラウザをサポートします。Safari でこれまで使用できなかった WebGrid のリッチな機能が Web ブラウザに実装されています。
