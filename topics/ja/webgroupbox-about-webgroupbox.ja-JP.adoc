= WebGroupBox について

WebGroupBox は、ASP.NET Web フォームのコンテナ コントロールです。これは標準 ASP.NET パネルと同じように機能しますが、クライアントおよびサーバー サイドの機能が追加され、表示やスタイリングのカスタマイズ機能が拡張されています。

WebGroupBox で提供される基本機能のリストを次に示します。

* ヘッダー テキスト/ヘッダー表示

WebGroupBox により、コンテナのヘッダー エリアにラベルが付けられます。ヘッダー ラベルがついたコンテナを使用することによって、すべての Web ページにおいてコントロールの論理グループ化が楽になり、整理がよくなります。

もちろん、この領域内のコントロールを簡単にスタイル設定したりカスタマイズしたりできます。単純な境界線と背景色を使用できます。また、背景画像を使用してパネル上にグラデーションやテクスチャの外観に仕上げることもできます。

image::Images\WebPanel_and_WebGroupBox_Overview_01.png[]

* 強化されたスタイル

WebGroupBox には、デザイナ/開発者が簡単に見栄えのあるユーザー インターフェイスを作成できるようにするスタイルおよび表示形式オブジェクトが豊富にあります。

また、スタイルは、含まれるすべてのコントロールに連鎖的に適用されるので、WebGroupBox により生成されるコントロール グループは、自動的に共通の前景色やフォント ファミリなどのスタイル設定を取ります。
