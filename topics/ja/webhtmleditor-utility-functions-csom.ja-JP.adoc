= WebHtmlEditor ユーティリティ関数

ここで説明するユーティリティ関数は、WebHtmlEditor コントロールの JavaScript コードで使用されます。ユーティリティ関数を提供し解説するのは、開発者ができるだけ多くのロジックをクライアント マシン上に実装することで、パフォーマンスと機能性の両面で性能を引き出せるようにすることを意図しています。

*function iged_getById(id)*

この関数は、WebHtmlEditor コントロールで使用します。

var editor = iged_getById().
