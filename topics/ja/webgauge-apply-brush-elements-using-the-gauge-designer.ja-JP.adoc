= ゲージ デザイナを使用してブラシ エレメントを適用

ゲージ デザイナの [外観] タブで、左のドロップダウン リストは使用可能なブラシのタイプが格納されています。選択するブラシ タイプによって、隣接するドロップダウン リストには選択されたブラシ タイプに使用可能な色のプロパティが格納されます。

ゲージ デザイナから使用可能なさまざまなブラシ エレメントがあります。ブラシのタイプと色を指定することによって、プロパティのルック アンド フィールを劇的に変更することができます。[外観] タブを使用すると、現在選択されているプロパティのブラシのタイプと色を簡単に変更できます。
