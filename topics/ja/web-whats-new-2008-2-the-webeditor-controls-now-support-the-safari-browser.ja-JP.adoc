= WebEditor コントロールは Safari ブラウザをサポート

{ProductName} 2008 Volume 2 から開始しましたが、すべての Web エディタは Apple® Safari™ Web ブラウザを完全にサポートします。これらのエディタの広範かつ多様な機能を Safari ブラウザで完全に使用可能となりました。
