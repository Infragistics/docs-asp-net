= 曜日ヘッダを非表示にする

デフォルトで WebMonthView はキャプション ヘッダの下の右上にヘッダを表示します。このヘッダには、 pick:[asp-net="link:{ApiPlatform}webui.webschedule{ApiVersion}~infragistics.webui.webschedule.webmonthview~dayofweekheaderformat.html[DayofWeekHeaderFormat]"] プロパティを設定して指定したフォーマットで曜日が表示されます。 

pick:[asp-net="link:{ApiPlatform}webui.webschedule{ApiVersion}~infragistics.webui.webschedule.webmonthview~dayofweekheadersvisible.html[DayofWeekHeadersVisible]"] pick:[asp-net="link:{ApiPlatform}webui.webschedule{ApiVersion}~infragistics.webui.webschedule.webmonthview~dayofweekheadersvisible.html[DayofWeekHeadersVisible]"] プロパティを設定することによって、希望する場合にはこのヘッダを非表示にすることができます。以下のコードはこの方法を示しています。

image::images\WebSchedule_How_to_Hide_the_Day_of_the_Week_Header_01.png[]

*Visual Basic の場合：*

----
Private Sub Page_Load(ByVal sender As System.Object, ByVal e _
  As System.EventArgs) Handles MyBase.Load
	Me.WebMonthView1.DayOfWeekHeadersVisible = False
End Sub
----

*C# の場合：*

----
private void Page_Load(object sender, System.EventArgs e)
{
	this.WebMonthView1.DayOfWeekHeadersVisible = false;
}
----
