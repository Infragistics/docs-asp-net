= ラベル ペイン

[ラベル] ペインによって、ゲージ上の注釈のテキストと回転角度を設定できます。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.labelappearance~formatstring.html[フォーマット文字列]"]  -- この値は注釈のテキストを設定します。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.labelappearance~rotationangle.html[回転角度]"]  -- この値を 0 から 100 の整数に設定します。この値は注釈の回転角度を決定します。
