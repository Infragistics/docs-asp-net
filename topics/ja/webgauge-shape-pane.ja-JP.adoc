= 形状ペイン

[形状] ペインは、目盛盤の形状を決定します。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.dialappearance~shape.html[形状]"]  -- この値は、目盛盤が楕円形か矩形かを指定します。

pick:[asp-net="link:{ApiPlatform}webui.ultrawebgauge{ApiVersion}~infragistics.ultragauge.resources.dialappearance~expansion.html[展開]"]  -- この値を -999 から 99 の値に設定します。
