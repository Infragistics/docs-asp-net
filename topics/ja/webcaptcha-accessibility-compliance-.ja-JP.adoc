= WebCaptcha アクセシビリティの遵守

弊社の ASP.NET コントロールおよびコンポーネントはすべて、1973 年のリハビリテーション法第 508 条第 1194 部 22 条を遵守しています。以下は、コントロールに関係する第 1194 部 22 条の特定の規則と各規則を遵守する方法を含んだ表です。場合によって、プロパティを設定する必要があります。それ以外の場合には、コントロールがすべての作業を実行するため、ユーザーは何もする必要がありません。

[options="header", cols="a,a"]
|====
|規則|規則を遵守する方法

|(a)
|主要な機能を持つすべての IMG タグは "alt" 属性プロパティが設定されるか、alt 属性値の設定が可能です。

|(b)
|WebCaptcha コンポーネントは、使用されている辞書モードに基づいて、ランダムに生成されるテキスト コード、ワード、または質問の回答を綴る機能をサポートします。

|(c)
|規則 A と B で説明されている属性のために、コントロールのマークアップは関連付けられたスタイル シートを必要とすることなく読み取り可能です。

|(d)
|コントロールのスクリプトがページ マークアップを変更している時、規則 A と C で説明されている属性のために新しく作成されたコンテンツはスクリーン リーダーに準拠します。

|(n)
|TAB キーを使用することによって、コントロールとその要素をナビゲートできます。

|====
