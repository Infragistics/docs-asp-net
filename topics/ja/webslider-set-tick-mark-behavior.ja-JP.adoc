= 目盛り動作を設定

エンド ユーザーが WebSlider™ の目盛またはラベルをクリックまたはダブルクリックした時の動作を制御できます。

目盛とラベルのクリック アクションを定義するために Tickmarks オブジェクトによって公開されるいくつかのプロパティを設定できます。

*  pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.editorcontrols.slidertickmarks~labelclickaction.html[LabelClickAction]"]  - エンド ユーザーがラベルをクリックした時の動作を定義します。
*  pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.editorcontrols.slidertickmarks~labeldoubleclickaction.html[LabelDoubleClickAction]"]  - エンド ユーザーがラベルをダブルクリックした時の動作を定義します。
*  pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.editorcontrols.slidertickmarks~tickmarkclickaction.html[TickmarkClickAction]"]  - エンド ユーザーが目盛をクリックした時の動作を定義します。
*  pick:[asp-net="link:{ApiPlatform}web{ApiVersion}~infragistics.web.ui.editorcontrols.slidertickmarks~tickmarkdoubleclickaction.html[TickmarkDoubleClickAction]"]  - エンド ユーザーが目盛をダブルクリックした時の動作を定義します。

以下の例のコードは、ダブルクリックした時につまみが目盛に移動する方法を示します。

image::images/WebSlider_Customize_Appearance_of_Tick_Marks_01.png[]

*Visual Basic の場合：*

----
Me.WebSlider1.Tickmarks.TickmarkClickAction = _
Infragistics.Web.UI.EditorControls.SliderTickmarkClickAction.MoveToTick
----

*C# の場合：*

----
this.WebSlider1.Tickmarks.TickmarkClickAction = Infragistics.Web.UI.EditorControls.SliderTickmarkClickAction.MoveToTick;
----
