= クリック イベントの処理

以下のコードは WebImageButton™ のクライアント側クリック イベントを処理する方法を示します。

*HTML の場合:*

----
<igtxt:WebImageButton ID="WebImageButton1" runat="server">
	<ClientSideEvents Click="WebImageButton1_Click" />
</igtxt:WebImageButton>
----

*JavaScript の場合*

----
function WebImageButton1_Click(oButton, oEvent){
    // スペース キーで起動した場合、クリックをキャンセル
    if (Event.action == 1)
        Event.cancel = true;
    // Enter キーで起動した場合、クリックをキャンセル
    if (Event.action == 2)
        Event.cancel = true;
    // Access キーで起動した場合、クリックをキャンセル
    if (Event.action == 3)
        Event.cancel = true;
    // マウス ボタンで起動した場合、クリックをキャンセル
    if (Event.action == 0)
        return;
}
----
