= WebCombo は Safari ブラウザをサポート

{ProductName} 2008 Volume 2 から開始しましたが、WebCombo™ コントロールは Apple® Safari™ Web ブラウザをサポートします。Safari でこれまで使用できなかった WebCombo のリッチな機能は今回ブラウザに実装されました。
