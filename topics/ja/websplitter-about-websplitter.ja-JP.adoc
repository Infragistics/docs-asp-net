= WebSplitter について

WebSplitter™ コントロールは、Web アプリケーション内でコンテンツを分離および整理するための優れた方法を提供します。WebSplitter はコンテンツ ペイン内に含まれているコンテンツをリサイズ、展開、縮小する機能をエンド ユーザーに提供します。Web アプリケーションをデザインする方法によっては、WebSplitter の方向を Horizontal または Vertical に変更することができます。垂直に向けると、コンテンツ ペインはひとつのペインの左右、つまりあるペインの横に配置されます。上から下までペインの間にスプリット バーが配置されます。スプリット バーを左右にクリック アンド ドラッグすると、ペインの幅を調整することができます。水平方向に向けると、ペインは上から下に配置されます。スプリット バーもペイン間に配置され、左から右に並びます。

以下の画像は、3 つのペインで構成された WebSplitter コントロールを示します。この特定の事例で、各ペインは URL にリンクされます。エンド ユーザーは、コンテンツをリサイズするためにスプリット バーをクリック アンド ドラッグすることができます。

image::images/WebSplitter_About_WebSplitter_01.png[]

WebSplitter は以下の機能をサポートします。

* 方向設定 - スプリット バーを水平方向または垂直方向に配置できます。
* リサイズの動作 - エンド ユーザーがスプリット バーをクリック アンド ドラッグするときに、マウスをドラッグするリアルタイムでコンテンツ ペインをリサイズできるように WebSplitter を構成できます。またはドラッグが完了してエンド ユーザーがマウスをリリースする後までリサイズを遅延することができます。
* コンテンツの展開 / 縮小 - スプリット バーはボタンを表示できます。このボタンはクリックされると、ボタンが表すコンテンツ ペインを展開または縮小することができます。
* コンテンツ ペイン - コンテンツ ペインのテンプレート内に任意のコンテンツを配置できます。これは、ユーザー コントロール、Web サーバー コントロール、またはユーザー インタフェースに必要なその他すべてを意味します。コンテンツ ペインは、さまざまな Web リソースからコンテンツを表示することができる URL も受け付けます。
